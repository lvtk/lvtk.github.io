◆ schedule_work

WorkerSchedule schedule_work

Use this to schedule a job with the host.

This is a function object so it can be called directly.

Example:

void run (uint32_t nframes) {
// ...
schedule_work (strlen ("non_rt_job"), "non_rt_job"));
// ...
}

The above code would result in work being called where "non_rt_job" is the data parameter
