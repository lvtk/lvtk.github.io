Detailed Description

template<class S, template< class > class... E>
class lvtk::UIInstance< S, E >

UI Instance.

Inherrit this when making an LV2 UI. The first template parameter is the type of your super class. The rest are feature mixins.

Examples
volume_ui.cpp.
