Detailed Description

template<class S, template< class > class... E>
class lvtk::UI< S, E >

UI Instance.

Inherrit this when making an LV2 UI. The first template parameter is the type of your super class. The rest are Extension mixins.

Template Parameters
SYour super class
EList of Extension mixins
