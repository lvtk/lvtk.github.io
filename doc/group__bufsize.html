Detailed Description

Access to buffer information.

The BufSize extension scans for buffer information provided by the host, and populates a BufferDetails accordingly. To use it, add BufSize to the template parameters of your Plugin and call buffer_details() for buffer info

class MyPlug : public lvtk::Plugin<MyPlug, lvtk::BufSize> {
public:
MyPlug (lvtk::Args& args) : lvtk::Plugin (args) {
const auto& details = buffer_details();
// setup plugin using details
}
};
