LV2 Toolkit

Introduction

These documents describe some C++ classes that may be of use if you want to write LV2 plugins in C++. They implement most of the boilerplate code so you only have to write the parts that matter, and hide the low-level C API.

Interfaces
For both the Instance and the ui::Instance class there are other helper classes called interfaces that you can use to add extra functionality to your plugin or UI, for example support for LV2 extensions.

Versioning
This library is available as a header only, thus ABI stability is not an issue. The API will be stable between major version bumps, at which the pkg-config name would change to prevent plugins from building against an incompatible version.
